3. Data transformation


A lambda function was applied in the 'age', 'income' and 'children' fields to replace negative values with 'NaN'.




Missing values were replaced with the median in the case of 'Age', 'Income', and 'Children', because these columns contain numeric values, and the median represents a central value that is less affected by outliers.

For 'Gender' and 'City', missing values were replaced with the mode, as these columns contain non-numeric values, and the mode represents the most frequent category.





A dictionary education_mapping is created to correct or standardize values in the Education_Level column. For example, "Bachelors" is corrected to "Bachelor", "pHd" is standardized to "PhD", and "no education" is replaced with "NE". The replace() method is then used to apply these mappings to the entire column.

After mapping, any missing values (NaN) in the Education_Level column are filled with "NE" (no education) using the fillna() method.

6. Brief Analysis.


a) Age:

Mean: 56.88 years, which is relatively high. This suggests that the dataset might be skewed towards older individuals.

Standard Deviation: 38.63 years, indicating considerable variation in ages.

Range: Ages vary from 1 to 200 years, with a very high maximum age suggesting possible outliers or data entry errors.

Percentiles: The 25th percentile is 28 years, the 50th percentile (median) is 53 years, and the 75th percentile is 78 years, indicating a spread with a substantial number of older individuals.

b) Income:

Mean: $65,778.29, which indicates the average income level in this dataset.

Standard Deviation: $33,442.61, showing high variability in income.

Range: Income ranges from $20,001 to $199,994, suggesting a wide income distribution.

Percentiles: The 25th percentile is $42,128.75, the median is $62,099, and the 75th percentile is $82,207.25. This shows that a significant portion of the dataset has incomes clustered around these values.

c) Height:

Mean: 1.75 meters, which is typical for human height.

Standard Deviation: 0.14 meters, showing relatively low variability.

Range: Heights range from 1.50 to 2.00 meters, which is within a reasonable human range.

Percentiles:The 25th percentile is 1.63 meters, the median is 1.75 meters, and the 75th percentile is 1.87 meters, indicating a fairly standard distribution of height values.

d) Children:

Mean: 2.44 children, indicating that on average, individuals have slightly more than two children.

Standard Deviation: 1.63, showing some variability in the number of children.

Range: From 0 to 5 children, which is a reasonable range for family size.

Percentiles: The 25th percentile is 1 child, the median is 2 children, and the 75th percentile is 4 children, suggesting that a significant number of individuals have a moderate to large number of children.

Overall:

The data seems to reflect a realistic distribution of ages, incomes, heights, and number of children, though the extremely high age value might warrant further investigation. The variability in income and number of children indicates diverse socio-economic backgrounds.
